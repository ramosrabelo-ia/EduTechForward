# Plano de Ação Rápido de Governança

A proposta é centralizar a documentação da EduTech Forward em um repositório único, reduzindo a perda de memória técnica e aumentando a transparência.

## Problemas encontrados

1. Dados descentralizados entre áreas.
2. Processo de cancelamento sem rastreabilidade.
3. Documentação técnica solta em Google Drive.
4. Ausência de dicionário de dados.

## Diretrizes

- Criar Single Source of Truth.
- Padronizar nomes e conceitos.
- Definir donos do processo e dos dados.
- Criar dicionário de dados.
- Controlar alterações via branch, Pull Request e revisão.
