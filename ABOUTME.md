# Sobre o Projeto

Este repositório apresenta uma proposta prática de governança de dados para a **EduTech Forward**, uma edtech em crescimento que enfrenta desafios relacionados à descentralização de dados, ausência de padronização, perda de memória técnica e baixa rastreabilidade no processo de cancelamento de matrícula.

A solução foi estruturada como uma primeira entrega de consultoria da **DataVault Soluções**, com foco em organizar informações críticas, definir responsabilidades, criar um dicionário de dados, propor padrões de nomenclatura e mitigar riscos relacionados à LGPD.

## Objetivo

Criar uma estrutura simples, clara e auditável para apoiar a governança de dados da EduTech Forward, usando o GitHub como ambiente central de documentação, versionamento e transparência.

## Principais entregas

- Mapa de atores e responsabilidades.
- Dicionário de dados do processo de cancelamento.
- Estrutura de repositório baseada em Single Source of Truth.
- Padrão de nomenclatura para tabelas e metadados.
- Análise de riscos LGPD e transparência.
- Página visual explicativa do projeto.

## Valor gerado

A proposta reduz silos entre áreas, melhora a comunicação entre Suporte, Financeiro, Vendas e Produto, aumenta a rastreabilidade das decisões e cria uma base mais segura para o uso de dados pessoais dos alunos.

## Repositório

Este projeto foi desenvolvido para fins acadêmicos na FIAP, na disciplina **Data Ethics, Governance & Security in the AI Age**.
