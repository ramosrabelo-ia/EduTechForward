# EduTech Forward — Plano de Governança de Dados

Projeto desenvolvido para o Checkpoint 3 da disciplina Data Ethics, Governance & Security in the AI Age.

https://ramosrabelo-ia.github.io/EduTechForward/

## Contexto

A EduTech Forward enfrenta dados descentralizados, ausência de dicionário de dados, perda de memória técnica e falta de rastreabilidade no processo de cancelamento de matrícula.

## Objetivo

Criar uma primeira estrutura de governança com foco em Single Source of Truth, dicionário de dados, rastreabilidade, LGPD, controle de acesso e padronização de metadados.

## Estrutura

- documentacao: relatórios, decisões e plano de ação.
- dados: bases mockadas e evidências.
- dicionario_dados: dicionário, glossário e padrões.
- processos: atores, responsabilidades e objetos de dados.
- seguranca_lgpd: riscos, matriz de acesso e transparência.
- fontes: página do projeto em HTML, CSS e JavaScript.
